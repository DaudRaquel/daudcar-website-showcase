## 🎨 Design System

### Cores
- **Primária:** `#000000` (Preto)
- **Secundária:** `#FFD700` (Dourado)
- **Acento:** `#FFA500` (Laranja)
- **Texto:** `#333333` (Cinza escuro)

### Tipografia
- **Principal:** 'Segoe UI', Tahoma, Geneva, Verdana
- **Títulos:** 4rem → 1.6rem (Responsivo)
- **Corpo:** 1.1rem → 0.9rem (Mobile)

### Breakpoints
- 🖥️ **Desktop:** 1024px+
- 💻 **Tablet:** 768px - 1023px
- 📱 **Mobile:** 480px - 767px
- 📱 **Small:** 360px - 479px

## ⚡ Performance

- 🚀 **First Contentful Paint:** ~1.2s
- 📦 **Bundle Size:** ~65KB (gzipped)
- 🟢 **Lighthouse Score:** 95+
- 📱 **Mobile Optimized**

## 🔧 Como Usar

### Pré-requisitos
- Node.js 16+
- npm ou yarn

### Instalação

```bash
# Clonar repositório
git clone https://github.com/seu-usuario/daudcar-website-showcase.git

# Entrar na pasta
cd daudcar-website-showcase

# Instalar dependências
npm install

# Iniciar desenvolvimento
npm start
```

### Scripts Disponíveis

```bash
npm start          # Servidor de desenvolvimento
npm run build      # Build para produção
npm test           # Executar testes
npm run eject      # Eject (irreversível)
```

## 🎯 Personalização

### Alterar Informações

1. **Logo:** Substitua `public/logo.png`
2. **Cores:** Edite `src/MainPage.css`
3. **Textos:** Modifique `src/MainPage.js`
4. **Contato:** Configure em `src/MainPage.js`

### Adicionar Novos Serviços

```javascript
// Em src/MainPage.js
const services = [
  {
    icon: 'fa-novo-servico',
    title: 'Novo Serviço',
    description: 'Descrição do novo serviço',
    features: ['Feature 1', 'Feature 2'],
    price: 'Consulte valores'
  }
];
```
